Manual Técnico

1. Introducción

Nombre del Proyecto: Sistema de Gestión de Reparaciones de Taller.

Descripción: Este sistema permite gestionar clientes, órdenes, roles, remisiones y métricas de un taller de reparaciones.

Tecnologías Utilizadas:

  • PHP (Backend)
  • MySQL (Base de datos)
  • HTML, CSS, JavaScript (Frontend)
  • Librería FPDF para generación de documentos PDF.

2. Estructura del Proyecto

Directorio Principal:

  • index.php: Archivo principal para manejar el enrutamiento.
  • manifest.json: Archivo de configuración.
  • README.md: Documentación básica del proyecto.

Carpetas:

  • assets/: Contiene recursos como imágenes, estilos CSS, scripts JS y la base de datos.
  • controllers/: Controladores para manejar la lógica de negocio.
  • models/: Modelos para interactuar con la base de datos.
  • views/: Vistas para la interfaz de usuario.

3. Configuración del Entorno

Requisitos:

  • Servidor web (XAMPP, WAMP, etc.).
  • PHP 7.4 o superior.
  • MySQL.

Configuración de la Base de Datos:

  • Importar el archivo baseDatos.sql ubicado en assets/bd/ a MySQL.
  • Configurar las credenciales de conexión en index.php:
$mysqli = new mysqli('localhost', 'root', '', 'reparaciones_taller');

4. Descripción de los Componentes

Controladores:

  • LoginController.php: Maneja la autenticación de usuarios.
  • ClienteController.php: Gestiona las operaciones relacionadas con clientes.
  • OrdenController.php: Gestiona las órdenes de reparación.
  • RolController.php: Administra los roles y permisos.
  • RemisionController.php: Genera y gestiona remisiones.

Modelos:

  • UsuarioModel.php: Modelo para usuarios.
  • ClienteModel.php: Modelo para clientes.
  • OrdenModel.php: Modelo para órdenes.
  • RolModel.php: Modelo para roles.
  • RemisionModel.php: Modelo para remisiones.

5. Flujo del Sistema

El archivo index.php actúa como enrutador principal.

Las acciones se manejan mediante el parámetro action en la URL.

Ejemplo: index.php?action=login muestra la página de inicio de sesión.

6. Mantenimiento

Errores: Activar la visualización de errores en index.php:

ini_set('display_errors', 1);
error_reporting(E_ALL);

Actualización de Dependencias: Asegurarse de que las librerías externas como FPDF estén actualizadas.
