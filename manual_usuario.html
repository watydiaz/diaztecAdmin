Manual de Usuario

1. Introducción

Propósito: Este manual guía al usuario en el uso del sistema de gestión de reparaciones.

Audiencia: Usuarios finales como administradores y técnicos del taller.

2. Acceso al Sistema

URL: Acceder al sistema mediante el navegador en http://localhost/diaztecAdmin/.

Inicio de Sesión:

  • Ingresar correo electrónico y contraseña.
  • Presionar el botón "Iniciar Sesión".

3. Funcionalidades Principales

Gestión de Clientes:

  • Navegar a la sección "Clientes".
  • Agregar, editar o eliminar clientes.

Gestión de Órdenes:

  • Navegar a la sección "Órdenes".
  • Crear nuevas órdenes, actualizarlas o eliminarlas.

Gestión de Roles:

  • Navegar a la sección "Roles".
  • Asignar o eliminar permisos a roles.

Generación de Remisiones:

  • Navegar a la sección "Remisiones".
  • Generar remisiones para órdenes específicas.

Visualización de Métricas:

  • Navegar a la sección "Dashboard".
  • Consultar métricas clave del taller.

4. Navegación

Menú Principal:

  • Opciones disponibles: Login, Dashboard, Clientes, Órdenes, Roles, Remisiones.

Acciones:

  • Botones para agregar, editar o eliminar registros.
  • Formularios para ingresar datos.

5. Resolución de Problemas

No puedo iniciar sesión:

  • Verificar que las credenciales sean correctas.
  • Contactar al administrador para restablecer la contraseña.

Error al agregar un cliente:

  • Asegurarse de que todos los campos obligatorios estén completos.
  • Verificar la conexión a la base de datos.

6. Contacto

Para soporte técnico, contactar al administrador del sistema.
